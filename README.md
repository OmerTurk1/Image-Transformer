# Image-Transformer
Turn Images Into Any Image

## Input Parameters
- source_photo_name: the photo is the landscape of the new photo.
- target_photo_name: the photo is the pixel sources of the new photo.
- org_photos_folder: where do you store original photos (exp. "photos/original/")
- res_photos_folder: where do you store resulted photos (exp. "photos/results/")
